Installation
################


Pypi
**********************

.. code-block:: console

    # Install from Pypi:
    pip install pypiplot

    # Force update to latest version
    pip install -U pypiplot


Github source
************************************

.. code-block:: console

    # Install directly from github
    pip install git+https://github.com/erdogant/pypiplot


Uninstalling
################


Remove installation
**********************

Note that the removal of the environment will also remove the ``pypiplot`` installation.

.. code-block:: console

    # Install from Pypi:
    pip uninstall pypiplot
