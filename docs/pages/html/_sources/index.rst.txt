-----------------------------------


``pypiplot`` is a Python package to count and plot the number of downloads from Pypi.


-----------------------------------

.. note::
	**Your ❤️ is important to keep maintaining this package.** You can `support <https://erdogant.github.io/pypiplot/pages/html/Documentation.html>`_ in various ways, have a look at the `sponser page <https://erdogant.github.io/pypiplot/pages/html/Documentation.html>`_.
	Report bugs, issues and feature extensions at `github <https://github.com/erdogant/pypiplot/>`_ page.

	.. code-block:: console

	   pip install pypiplot
